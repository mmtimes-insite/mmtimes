Gesture-Controlled Drone Workshop

About the Gesture-Controlled Drone Workshop

The Gesture-Controlled Drone Workshop is a fully hands-on program where participants learn how drones can be controlled using human hand gestures. This workshop covers gesture recognition, sensor integration, IMU processing, and drone command mapping. Ideal for students, hobbyists, robotics clubs, tech enthusiasts, and beginners exploring smart UAV control systems. By the end of the workshop, each participant will build, configure, and test a drone that responds interactively to real-time hand movements.
