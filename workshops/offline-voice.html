Offline Voice Command Workshop

About the Offline Voice Command Workshop

The Offline Voice Command Workshop is a practical, hands-on training designed to teach participants how offline speech recognition systems work and how voice commands can be used to control electronic devices without internet connectivity. This workshop is perfect for students, hobbyists, automation enthusiasts, and beginners interested in AI-driven embedded systems. By the end of the workshop, each participant will build and program a fully functional offline voice-controlled automation system using tools like PocketSphinx or Vosk.
